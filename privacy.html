Privacy Policy

Last updated:

Summary: OsynX Bridge is a Google Sheets™ add-on that connects user spreadsheets to their Samsara™ account. The add-on runs entirely within the user’s Google account environment and does not transmit or store data on OsynX servers.

1. Data Accessed

The add-on accesses the following Google user data via OAuth scopes:

  • Google Sheets data to read and write cell values in the active spreadsheet.
  • Basic Google profile information (name, email) for identifying the signed-in user.
  • Apps Script container and locale settings to render sidebar UI and format data correctly.

2. Data Usage

All accessed data is used solely to:

  • Display fleet information from Samsara inside the user’s spreadsheet.
  • Store user configuration (e.g. API token, Org ID) in that spreadsheet’s properties.
  • Render the add-on’s sidebar and update rows when triggered by the user.

No Google user data is used for advertising, analytics, or sold to third parties.

3. Data Sharing

OsynX Bridge does not share any Google user data with external parties. API calls to Samsara™ occur directly between the user’s Sheet and Samsara’s API endpoints over HTTPS.

4. Data Storage & Protection

No data from Google Sheets or user profiles is transmitted or stored on OsynX servers. Configuration values entered by the user are stored locally within that spreadsheet’s properties, protected by Google’s own account security infrastructure.

5. Data Retention & Deletion

OsynX Bridge does not retain user data. All information remains in the user’s spreadsheet until they delete or modify it. Users can revoke the add-on’s access anytime via their Google Account permissions.

6. Contact

For privacy questions, contact us at ochyro@osynx.com.
