Quick start

Setup checklist

Install
Install the Sheets™ add-on (unlisted link).
Token
Create a read-only Samsara API token and copy your Org ID.
Configure
Open the add-on sidebar → paste token & Org ID → Save.
Test
Pick an asset in a row and click Update. Use Refresh All for batch.
FAQ

Common questions

Where is my token stored?
In your Google Sheet’s properties—never on OsynX servers.

What permissions are needed?
Read-only access to the Samsara endpoints you use. The add-on itself uses standard Sheets™ and Apps Script scopes.

I see “unverified app”.
If you’re testing early builds, that’s normal. For public use, we’re completing OAuth verification.
